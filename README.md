# Skript Projects
A collection of skript projects I have made.
-----------------
**fly.sk**
- Command: /fly
- Description: Toggles on and off flight
**flyspeed.sk**
- Command: /flyspeed [<number>]
- Description: Change your flight speed with a value from 1 to 10
**heal.sk**
- Command: /feed
- Description: Fills hunger bar
**feed.sk**
- Command: /heal
- Description: Fills health bar
**toggledamage.sk**
- Command: /toggledamage
- Description: Toggles on and off player damage
**autowelcome.sk**
- Command: None, Works in background
- Description: Sends a global message when a new user joins the server, sends a welcome back message for users that have joined before
